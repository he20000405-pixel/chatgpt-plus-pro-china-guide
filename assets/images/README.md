# 图片目录说明

首版不公开展示模糊截图。后续请把经过裁剪、隐私检查和清晰度优化的真实图片放在本目录。

建议文件名：

- `chatgpt-plans.webp`：`/chatgpt` 套餐选择区域；
- `chatgpt-checkout.webp`：支付宝/微信付款与卡密说明；
- `verify-card-key.webp`：`/verify` 卡密验证区域；
- `verify-session.webp`：session 提交与账号确认区域；
- `chatgpt-membership-confirmation.webp`：充值后官方会员状态确认页。

发布前要求：

1. 只使用真实页面截图，不伪造成交、评价或官方背书；
2. 遮盖卡密、session、邮箱、订单号和其他用户数据；
3. 建议宽度 1400 至 1800 像素，优先使用 WebP；
4. 图片加入后，在对应文章中取消 HTML 注释占位符并补充准确 alt 文本；
5. 页面价格变化后及时替换旧截图。
